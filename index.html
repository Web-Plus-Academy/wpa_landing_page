Overview of 6 months course

Mode: Online

Technologies covered
  • Frontend: HTML, CSS, JavaScript
  • Framework: ReactJS, TailwindCSS
  • Backend: NodeJS, ExpressJS, TypeScript.
  • Libraries: Mongoose, Sequelize, JWT, SocketIO, SMTP
  • SQL Database: PostgreSQL
  • NoSQL Database: MongoDB
  • Problem Solving: LeetCode

Course divided into 3 Semesters of 2 months each

Each Semester have 8 weeks Tasks , 8 Projects and One Major Project of that semister

At the end of the each Semester you will receive your Report card which contains

  1. Week tasks credits
  2. Attendance Percentage
  3. Sem-project credits
  4. Grade for the Semester

Daily "Problem of the Day" will be given to solve for enriching their problem solving skills.
